# Touchscreen settings + recovery

## Opšte postavke
Novi ekran **Opšte postavke** pokriva:
- naziv terminala
- lokaciju terminala
- idle timeout
- display brightness
- screensaver / blanking ponašanje
- kiosk mode info

## Guided recovery
Novi ekran **Guided Recovery** daje tri bloka:
- šta prvo provjeriti (startup)
- printer recovery
- network recovery

## Preporučeni terenski tok
1. Health screen
2. Ako nešto padne: `Recovery pomoć`
3. Ako je printer problem: `Setup printera` + test print po uređaju
4. Ako je mreža problem: `Mreža / Wi‑Fi`
5. Ako treba fino podešavanje uređaja: `Opšte postavke`
